งานบริการซ่อมบำรุง งานบริการโสตทัศนูปกรณ์
ขั้นตอน แบบฟอร์ม รหัสแบบฟอร์ม ระยะเวลา หมายเหตุ
1.รับแจ้งซ่อมจากฝ่ายต่างๆ แบบฟอร์มในระบบแจ้งซ่อม (E-Fxmc) MAT-QP-001 - ฝ่ายต่างๆ
2.ตรวจเช็ค / หาสาเหตุและแนวทางแก้ไขปัญหา แบบฟอร์มในระบบแจ้งซ่อม (E-Fxmc) MAT-QP-001 - ผู้ตรวจเช็คระบุข้อมูลในระบบ
3.มอบหมาย / เข้าตรวจเช็ค / ส่งซ่อม แบบฟอร์มในระบบแจ้งซ่อม (E-Fxmc) MAT-QP-001 - อาร์ต/รักษ์ ตรวจเช็คระบบและบันทึกมอบหมาย(หรือทำเอง)
4.รูปแบบการซ่อม แบบฟอร์มในระบบแจ้งซ่อม (E-Fxmc) MAT-QP-001 - ผู้ตรวจเช็คระบุข้อมูลในระบบ
    4.1 ซ่อมเอง ตรวจเช็คพร้อมกำหนดระยะเวลาการซ่อม - - -
    4.2 ส่งซ่อม / ติดต่อประสานงานกับร้านค้า ใบประมาณราคา/เอกสารจากร้านค้า - - -
      4.2.1 ทำเรื่องขออนุมัติ / ยืมเงินทดรองจ่าย หนังสือขออนุมัติ - - งานซ่อมบำรุงทำเรื่องขออนุมัติเมื่อทราบราคาซ่อมแล้ว
      4.2.2 ประสานงานกับร้านค้าเพื่อซ่อม โทรแจ้ง / ประสานรูปแบบการรับซ่อม MAT-FM-001 - เอกสารใบรับซ่อมจากร้านค้า
      4.2.3 รับงานซ่อมพร้อมทดสอบงานซ่อม ใบรับคืนงานซ่อม/ใบวางบิล MAT-FM-001 - เอกสารจากร้านค้า / แบบฟอร์มของสำนักฯ
      4.2.4 ชำระเงิน/เบิกจ่าย แบบฟอร์มยืมเงินทดรองจ่าย/เอกสารการตั้งเบิก แบบฟอร์มการเงิน - กรณีชำระเงินสดให้ยืมเงินทดรองจ่าย / หากวางบิลเพื่อตั้งเบิกใช้กระบวนการพัสดุต่อไป
5.ส่งมอบงาน แบบฟอร์มในระบบแจ้งซ่อม (E-Fxmc) MAT-QP-001 - การส่งมอบงานนี้รวมทั้งกรณีการซ่อมเอง
